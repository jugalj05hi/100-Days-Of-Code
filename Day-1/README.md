# Day-1: Getting started with sorting Algorithms
<ol>
    <li> Selection Sort </li>
        <ul>
            <li> Best case: O(n^2) </li>
            <li> Worst case: O(n^2) </li>
            <li> Average case: O(n^2) </li>
            <li> Space Complexity: O(1) </li>
            <li> Stable sort: No </li>
        </ul>
